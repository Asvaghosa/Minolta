○サンプルアプリについて
 本アプリはコンソールアプリです。
 各機種（CM-25cG/CM-M6/CM-26d CM-25d/CM-26dG/CM-23d）ごとにサンプルアプリを用意しております。
 また、作成環境ごとにも分かれております。
 ・C#環境でC#版SDKを使用する場合         ：SampleApp_CS
 ・C/C++環境でC/C++版SDKを使用するの場合 ：SampleApp_C
 ・C#環境でC/C++版SDKを使用する場合      ：SampleApp_CS_C

 サンプルコードは以下の3種類を用意しています。
 1：校正・測定・測定データ取得(PerformMeasurement)
 2：保存されている測定データの取得(ReadSampleData)
 3：基準データの書き込み(WriteTargetData)
 上記の動作させたい番号と測定器と接続されているCOMポート番号をコマンドラインの引数として入力します。
 例えば、校正・測定・測定データ取得したい場合、コマンドプロンプトで以下のように入力します。
 SampleApp 1 COM10
